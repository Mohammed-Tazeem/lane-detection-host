About Project

Road Lane Detection Project using python and Open CV library

The lines on the road that show us where the lanes are act as our constant reference for where to steer the vehicle. Naturally, one of the first things we would like to do in developing a self-driving car is to automatically detect lane lines using an algorithm. Our Goal is :

  • To creat Pipeline to detect the line segments in the image
  • Then average/extrapolate them.
  • Draw them onto the image for display.

Pipeline

The Overall Projects follows this pipeline for carrying the operation in this sequence.

Gray Scaling

Converting to grayscale for an 8-bit image, so each pixel can take 2^8 = 256 possible values. Hence, the pixel values range from 0 to 255

Gaussian Smoothing

For suppressing noise and spurious gradients by averaging

Canny Edge Detection

Edge detection algorithm that aims to achieve low error rates, good localization, and minimal response by detecting only existent edges with high accuracy and minimal noise

Region Masking

Region Masking involves selecting pixels within a specific region, such as a triangular area, for processing, while ignoring the rest of the image, to focus on areas like lane lines where they are expected to appear, using a mask to retain only the desired region.

Hough Transform

The Hough Transform converts lines from image space (x vs. y) to Hough space (m vs. b), allowing for the detection of lines by analyzing points in parameter space that represent all possible lines passing through each pixel in the image space.

Draw Lines

To Draw the line on the image based on the pipeline algorithm
